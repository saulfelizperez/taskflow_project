Frontend de TaskFlow encargado de la interfaz de usuario y consumo de la API REST.

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Fetch API

index.html

https://taskflow-project-l1d1.onrender.com

## Funcionalidades

- Listar tareas
- Crear tareas
- Editar tareas
- Eliminar tareas
- Actualización sin recargar página

---

## Flujo

1. Usuario interactúa
2. JS captura evento
3. Fetch a API
4. Backend responde JSON
5. UI se actualiza

---

## Funcionalidad principal

Frontend 100% dinámico consumiendo API REST.
